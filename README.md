BLACK TIME is a **professional OSINT tool** for **Termux and Linux**.  
It provides fast scanning for **Usernames, Websites, IPs, Ports, and Subdomains**.

---

## ⚡ Features

- ✅ Username scan across multiple platforms: GitHub, Instagram, Twitter, TikTok, YouTube  
- ✅ Website status check  
- ✅ IP lookup  
- ✅ Port scan (common ports: 21,22,80,443,8080)  
- ✅ Subdomain scan (www, mail, api, dev, ftp)  
- ✅ Multi-thread scanning for speed  
- ✅ Save results in TXT, JSON, or both  
- ✅ Colored CLI interface
